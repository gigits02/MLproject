## CUP â€“ Regression task

The `CUP/` directory contains all the material related to the CUP regression problem.

### ğŸ“ CUP_datasets/

This folder contains the **raw dataset files**, separated into:

* **training** files;
* **test** files.

These files are not modified directly and represent the base input for all experiments.

### ğŸ“ GridSearch/

This folder contains the code and outputs related to hyperparameter tuning:

* `grid.py`

  * performs a **grid search** over the main neural network hyperparameters;
  * stores the results (hyperparameter configurations and validation metrics) into output files.

* `sortGrid.py`

  * post-processes the grid search results;
  * sorts configurations by **increasing validation loss**, making model selection easier.

The files generated by these scripts are included in the folder.

### ğŸ“„ crossVal.py

Main script for training and evaluating the model using **k-fold cross-validation**.

Main functionalities:

* neural network definition;
* *from-scratch* training procedure;
* performance evaluation averaged across folds;
* support for hyperparameter assessment.

This script is intended for experimental analysis and model validation.

### ğŸ“„ finalTest.py

Script used for the **final evaluation** of the selected model.

Main steps:

* training the neural network on the full training set;
* testing on the CUP test dataset;
* generation of the final output file:

```
output_test.csv
```

containing the required predictions for submission.

## Requirements and execution

* Language: **Python 3**
* Libraries: only standard libraries (e.g. `math`, `random`, `csv`, `itertools`, `numpy` if allowed)
* No high-level machine learning frameworks are used.

Scripts can be executed directly from the terminal, for example:

```bash
python crossVal.py
python finalTest.py
python monk.py
```

(from their respective directories).
